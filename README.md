# 🚗 AutoTracker – Mert & Tobias

Auto-Handel Tracking App zur Verwaltung von gekauften und verkauften Fahrzeugen.

## Setup

1. **Node.js installieren** (falls nicht vorhanden): https://nodejs.org
2. **Terminal öffnen** und in den Projektordner navigieren
3. **Dependencies installieren:**
   ```bash
   npm install
   ```
4. **Server starten:**
   ```bash
   node server.js
   ```
5. **Browser öffnen:** http://localhost:3000

## Features

- 📊 Dashboard mit allen wichtigen Kennzahlen
- 🚗 Autos anlegen, bearbeiten, verkaufen, löschen
- 💸 Ausgaben pro Auto tracken (Reparatur, TÜV, Aufbereitung etc.)
- 💰 Pot-Verwaltung (gemeinsame Kasse)
- 👥 Erstattungs-Tracking (Mert & Tobias)
- 📈 Gewinn/Verlust pro Auto
- 📜 Status-Verlauf pro Auto

## Daten

Alle Daten werden lokal in `data/database.json` gespeichert.
